SocketChat

By Ahnaf Wafiq

A simple group chat app built by Ahnaf Wafiq. The app leverages Web Socket connections to deliver instant messaging experience. Users can enter in a username and chat in this global group chat. Messages are stored and delivered securely with powerful API and Database.
